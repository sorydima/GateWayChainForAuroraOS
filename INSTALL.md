# Installation Guide

## Prerequisites
- Aurora OS or Sailfish OS device
- Qt 5.x and QML
- CMake or qmake toolchain

## Build Instructions
```bash
git clone https://github.com/sorydima/GateWayChainForAuroraOS.git
cd GateWayChainForAuroraOS
qmake && make
./GateWayChain
```

## Notes
Ensure all Qt dependencies are installed for Aurora OS.
